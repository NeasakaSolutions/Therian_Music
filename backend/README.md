### Dependencias del backend:

- Express
```bash
npm install express
```

- Nodemon
```bash
npm i -D nodemon
```

- Typescript
```bash
npm i -D typescript ts-node
```

- Mongoose:
```bash
npm i mongoose
```

- Dotenv:
```bash
npm i dotenv
```

- Colors:
```bash
npm i colors
```

- Opcionales:
```bash
npm i --save @types/express
```
