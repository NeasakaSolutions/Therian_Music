# Frontend - Proyecto React

Este repositorio contiene el **frontend** del proyecto, desarrollado con **React** utilizando **TypeScript** y **SWC** para un mejor rendimiento en el proceso de compilación.

## Tecnologías utilizadas

- **Framework:** React
- **Lenguaje:** TypeScript
- **Compilador:** SWC
- **Gestor de paquetes:** npm / yarn / pnpm
- **Estilos:** CSS / Tailwind CSS / Styled Components _(según aplique)_

#  Therian Music – Frontend

Frontend de la aplicación **Therian Music**, una plataforma musical con una experiencia visual moderna, animada y enfocada en UI/UX premium.

El proyecto está construido con **React + TypeScript**, usando **Vite** y **Tailwind CSS**, e incluye un sistema de autenticación visual (Login / Register) con transiciones animadas, modo oscuro y efectos avanzados.

---

##  Stack Tecnológico

-  **React**
-  **TypeScript**
-  **Vite**
-  **Tailwind CSS**
-  **Dark / Light Mode**
-  **React Router DOM**
